처음 실행 방법
================

[방법 A] Python 없이 (데스크톱 앱)
1. dist\TasteLab 폴더 안의 TasteLab.exe 를 더블클릭합니다.
2. 작은 창이 뜨면 잠시 기다립니다.
3. 브라우저에서 http://127.0.0.1:8000 이 자동으로 열립니다.
4. 종료할 때는 TasteLab 창을 닫으세요.

[방법 B] 개발/소스 실행 (Python 필요)
1. 압축을 풉니다.
2. 폴더 안의 [앱실행.bat]을 더블클릭합니다.
3. 검은 창이 열리면 닫지 말고 기다립니다.
4. 브라우저에서 http://127.0.0.1:8000 이 자동으로 열립니다.

상단 메뉴
---------
- 영화 추천: 장르, 영화, 감독, 배우 기반 추천 + 영화 기반 음악 추천
- 음악 추천: 장르, 아티스트, 좋아하는 노래, 좋아하는 앨범 기반 추천

음악 추천 API 키
----------------
음악 추천을 쓰려면 .env.example 파일명을 .env로 바꾸고 아래처럼 입력하세요.

LASTFM_API_KEY=본인_LastFM_API_키

주의: 파일명이 .env.txt가 되면 안 됩니다.
